# AgmDirectionTest

Testing for agm-direction.

This project was generated by [Angular CLI](https://cli.angular.io/). 

## Installation

```bash
yarn install
```

## Build and pack

Root directory:

```bash
yarn pack:build
```

## Install from pack

```bash
cd playground
```

```bash
yarn add [path]my-library-[version].tgz -S
```

Example:

```bash
yarn add agm-direction-0.7.0.tgz -S
```

## Start

```bash
yarn start
```
